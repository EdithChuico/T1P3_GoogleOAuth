**Estudiante:** Edith Chuico  
**Proyecto:** Sistema de gestion de Productos con Northwind  
**Stack Tecnologico:** Node.js con Express, TypeScript, JWT, Google OAuth.  
**Base de datos seleccionada:** Northwind  
**Instrucciones básicas de ejecución**.
1. Es importante incialmente instalar las depedencias, en este caso con "npm install"  
2. Posterior a ello, es necesario configurar el archivo .env, para ello puede basarse en env.ejemplo
3. Después debe de ejecutar el script de la base de datos
4. Finalmente, para inicialziar en modo de desarrollo, debe de usar npm run dev
